## 🔑 API Keys管理

各プラットフォームのAPIキーは `.env` ファイルで管理されています。

```bash
PRINTIFY_API_KEY=...
SUZURI_ACCESS_TOKEN=...
```

詳細は [.env.example](.env.example) を参照してください。
